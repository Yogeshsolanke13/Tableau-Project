# Global Superstore Dashboard (Tableau)

## Project Overview
This project presents an interactive Tableau dashboard built using the Global Superstore dataset. The dashboard helps analyze sales, profit, shipping costs, product categories, and country-wise performance to support data-driven business decisions.

## Tools Used
- Tableau
- Microsoft Excel

## Dashboard Features
- Sales and Profit Analysis
- Country-wise Sales Performance
- Product Category Analysis
- Shipping Cost Analysis
- Year-wise Sales Trend
- Interactive Filters and Visualizations

## Key Insights
- Identified top-performing countries based on sales.
- Analyzed best-selling product categories.
- Observed year-wise sales growth trends.
- Compared shipping costs across different regions.
